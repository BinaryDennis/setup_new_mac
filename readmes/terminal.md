## Pimp your Terminal

1. Download color scheme for iTerm: http://iterm2colorschemes.com
2. iTerm2 → Preferences → Profiles → Colors → Color presets → Import. Then again, Color presets → Choose `Smoooth`
3. iTerm2 → Preferences → Profiles → Text → Change Font 
4.  iTerm2 → Preferences → Profiles → Text→ Cursor : ✓ Vertical Bar → Blinking cursor : ✓ ON
5.  iTerm2 → Preferences → Advanced → Maximum number of lines → 100.000
